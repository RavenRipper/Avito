Инструкция по запуску автоматизированных тестов

1.Клонировать репозиторий:

git clone <ссылка на репозиторий>

2.Установить Python (версии 3.8+) и перейти в директорию проекта.

3.Установить зависимости:

pip install -r requirements.txt  # если файл requirements.txt существует

4.Перейти в директорию с тестами:

cd tests

5.Запустить тесты:

pytest


6.Проверить результаты:

Успешные тесты: passed.

Неудачные тесты: failed с описанием ошибки.

Требования:

Python 3.8+.

Интернет-доступ для обращения к API (https://qa-internship.avito.com).
